python贪吃蛇小游戏

  • 安装下载python

    官网 https://www.python.org/

    下载地址 https://www.python.org/downloads/windows/

    下载后运行并安装。注意： 官网明确表示，3.5及以上版本不支持xp操作系统。要兼容更低操作系统的，使用低版本python。

  • 安装pip
    pip是python中用来安装其他包的工具。在python更早版本中，pip需要下载安装，请自行上网查找教程。在python3.7中，pip是随python捆绑安装，不需要另外安装。

    查看pip是否安装：
    查看 python安装目录\Scripts下是否有pip.exe、easy_install.exe等文件，有的话说明pip安装成功。

  • 安装pygame
    pygame是python的一个游戏库。官网：http://www.pygame.org，

    打开cmd，

    输入：
    python -m pip install pygame –user
    会自动下载安装pygame包，下载界面如图。安装成功会提示success，

    py -m pygame.examples.aliens 通过这条命令测试是否安装好

    现在可以开始使用pygame模块了 ：)
